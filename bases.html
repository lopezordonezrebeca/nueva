Tipos de base

Base líquida

Son aptas para cualquier tipo de piel, aunque es recomendable para pieles normales y secas. También son perfectas para unificar el tono y dejar un acabado natural. Existen las líquidas hidratantes y bases líquidas con acabado mate.

Ofrece varios tipos de cobertura, desde ligera a muy cubriente.

Podemos aplicarlo con los dedos, con esponjita o con brocha de base.

Base en crema

Es un maquillaje de cobertura media recomendable para pieles normales-secas. Deja un acabado mate aterciopelado muy natural.

No es recomendable que se utilicen en pieles grasas ya que no absorberán bien y el acabado desaparece fácilmente.

Se puede aplicar fácilmente con la yema de los dedos o con una esponjita.

Base en polvo

Esta base suelen ser de cobertura media-alta, se pueden aplicar con una simple esponjita.

Para aplicarlos es imprescindible una brocha específica.

son perfectos para quitar brillos, por lo que vienen genial para las pieles grasientas o las zonas brillantes que queramos tapar.

Se recomienda para pieles mixtas-grasas ya que puede resecar o marcar las pequeñas arrugas si tenemos la piel seca.

Base en barra

Es altamente cubriente y de larga duración, es recomendable para pieles mixtas y grasas.

Además también es útil como corrector, ideal para tapar marcas e imperfecciones.

Es perfecta para llevarla contigo a cualquier parte gracias a su pequeño formato.

Podemos aplicarlas directamente sobre la piel o aplicarla en los dedos y luego en el rostro. De esta forma podrás modular la cobertura del acabado y además incidir en algunas zonas.
